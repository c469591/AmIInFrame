## 功能特色

* 一键检查：按下快捷键即可知道当前是否在页框中
* 嵌套页框支持：可侦测多层嵌套的页框结构
* 显示页框名称：如果页框有名称，会一并朗读
* 多语言支持：繁体中文、简体中文、英文

## 快捷键

### 默认快捷键

* NVDA+Shift+M：检查当前焦点是否在页框中（与单键浏览跳到页框的 M 键呼应）

### 自定义快捷键

可通过 NVDA 的「输入手势」对话框自定义快捷键，在「我在页框里吗」分类下找到相关功能。

## 使用方法

### 基本使用

1. 在浏览网页时，按 `NVDA+Shift+M`
2. 插件会朗读当前是否在页框中
3. 如果在页框中，还会告诉你页框的名称（如果有的话）

### 朗读示例

* 不在页框中：「不在页框中」
* 在单个页框中：「在页框中」或「在页框中：{页框名称}」
* 在多层嵌套页框中：「在 2 层嵌套页框中」

## 什么是页框（iframe）？

页框是网页中嵌入的另一个网页，常见于：

* 广告区块
* 嵌入的视频播放器（如 YouTube）
* 社交媒体分享按钮
* 第三方登录表单
* 在线编辑器

当你在页框中浏览时，某些 NVDA 的导航功能可能会受限于该页框范围内，了解自己是否在页框中可以帮助你更好地导航网页。

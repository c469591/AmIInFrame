### 我在页框里吗 NVDA 插件

这是一个功能超简单的NVDA附加元件，功能只有一个:
检测当前光标停留的位置是否在页框（iframe）中。
会开发这个插件，只是为了更方便的进行网站无障碍检测，因为现在的NVDA只有跳转到页框和离开页框的功能，
当你使用网页搜索功能跳到某个位置的时候，没有一个简单的功能可以得知自己是否在页框里，这样会很难界定当前看到的内容，是网站本身的，还是从其他网站内嵌进来的。

#### (小补充) 到底什么是页框

页框就是在自己网站的某个网页里，动态嵌入其他网站的内容，
例如嵌入youtube、facebook贴文，这是很多台湾的政府网站惯用的用法，他们往往喜欢把一些重要消息公布在社交平台上，然后自己的政府网站懒得再重写一遍内容，就直接把facebook的贴文插入到自己的网页上。
所以很多时候使用nvda(屏幕阅读软件)的用户在浏览时，会看到一些乱七八糟的图片、链接、点赞数，这很多时候都是因为页框中嵌入了其他网站，而那些网站的设计没有考虑无障碍使用体验导致的结果。


# 我在頁框裡嗎 NVDA 插件

檢查當前焦點是否在頁框（iframe）中，以享受更好的網頁瀏覽體驗。

### 相容性

NVDA 2024.1 以上版本測試正常

## 功能特色

* 一鍵檢查：按下快捷鍵即可知道當前是否在頁框中
* 巢狀頁框支援：可偵測多層巢狀的頁框結構
* 顯示頁框名稱：如果頁框有名稱，會一併朗讀
* 多語言支援：繁體中文、簡體中文、英文

## 快捷鍵

### 預設快捷鍵

* NVDA+Shift+M：檢查當前焦點是否在頁框中（與單鍵瀏覽跳到頁框的 M 鍵呼應）

### 自定義快捷鍵

可透過 NVDA 的「輸入手勢」對話框自定義快捷鍵，在「我在頁框裡嗎」分類下找到相關功能。

## 使用方法

### 基本使用

1. 在瀏覽網頁時，按 `NVDA+Shift+M`
2. 插件會朗讀當前是否在頁框中
3. 如果在頁框中，還會告訴你頁框的名稱（如果有的話）

### 朗讀範例

* 不在頁框中：「不在頁框中」
* 在單個頁框中：「在頁框中」或「在頁框中：{頁框名稱}」
* 在多層巢狀頁框中：「在 2 層巢狀頁框中」

## 什麼是頁框（iframe）？

頁框是網頁中嵌入的另一個網頁，常見於：

* 廣告區塊
* 嵌入的影片播放器（如 YouTube）
* 社群媒體分享按鈕
* 第三方登入表單
* 線上編輯器

當你在頁框中瀏覽時，某些 NVDA 的導航功能可能會受限於該頁框範圍內，了解自己是否在頁框中可以幫助你更好地導航網頁。
